# Support

AgentContextKit is an early open source project.

## Where To Ask
Use public issues once a public repository exists.

Do not include:
- secrets
- private source code
- production config
- customer data
- private logs

## Security Reports
See `SECURITY.md`.

## Scope
Maintainers can help with:
- CLI usage
- config format
- false positives
- generated docs
- local package validation

Maintainers cannot guarantee:
- perfect secret detection
- full repository cleanup
- legal approval for public release
- NuGet publishing on behalf of users
