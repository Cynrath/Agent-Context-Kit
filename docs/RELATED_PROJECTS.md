# Related Projects

## Status
This is an initial catalog, reviewed on 2026-06-13 from official repositories and project documentation. It is not an endorsement, dependency list, or compatibility guarantee. License, privacy, CLI, and output contracts must be reviewed again before any interoperability implementation.

Offline labels:
- `Yes`: the primary listed workflow can process a local repository without uploading it or requiring a hosted API after installation.
- `Partial`: a useful local path exists, but model inference, database updates, collaboration, remote repositories, or major features may use network/API services.
- `No`: the relevant workflow requires a hosted service or API. No initial catalog entry is recommended in this class.
