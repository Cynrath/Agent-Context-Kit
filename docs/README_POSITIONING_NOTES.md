# README Positioning Notes

## Product Sentence
AgentContextKit prepares repositories for AI coding agents through local readiness checks, safe context files, task/handoff discipline, privacy-aware scanner outputs, and release evidence.

## Ecosystem Boundary
Related packers, graphs, search engines, scanners, SBOM tools, and documentation tools can complement a reviewed repository. AgentContextKit does not install or invoke them by default and does not claim to replace their specialized analysis.

## README Constraints
- Keep the ecosystem section to one short paragraph and three to five bullets.
- Link to evidence, workflows, positioning, and pipeline docs instead of embedding the matrix.
- State the no-install/no-invoke default explicitly.
- Avoid endorsements, popularity claims, integration-ready language, and release-ready claims.
- English and Turkish sections must carry the same trust-boundary meaning.

## Terms To Prefer
- local repository readiness
- privacy-first findings
- agent instructions and handoff
- optional external enrichment
- reviewed local workflow
- maintainer-gated release decision

## Terms To Avoid
- all-in-one replacement
- guaranteed secure
- 1.0 ready
- automatic integration
- fully offline for every related tool
- supported external adapter
